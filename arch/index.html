Arch Linux 安装部分 1.制作启动盘 去往Arch Linux官网下载iso镜像 使用任意制作启动盘的工具制作启动U盘 制作细节此处不做赘述 2.安装(UEFI MODE) U盘插在USB插口后，选择U盘进入(不同电脑品牌，按键不同) 2.1更改字体 进入后如果感觉字体过小，则可以更改字体 setfont /usr/share/kbd/consolefonts/LatGrkCyr-12x22.psfu.gz 2.2无线联网 使用ip link查询无线网卡接口 若无线网卡接口为DOWN,则通过ip link set YOUR_PORT up命令启用端口 通过iwlist YOUR_PORT scan | grep ESSID命令扫描无线 wpa_passphrase YOUR_WIFI_NAME YOUR_WIFI_PASSWORD > internet.conf将wifi生成对应配置文件 wpa_supplicant -c internet.conf -i YOUR_PORT &连接无线 dhcpcd &若为有线连接，则只需此一条命令 2.3设置时钟 timedatectl set-ntp true 2.4分区(fdisk) 使用fdisk -l查看磁盘列表 fdisk /dev/nvme0n1 ...... fdisk用法见arch wiki mkfs.fat -F32 /dev/nvme0n1p1此处nvme0n1p1为引导分区 mkfs.ext4 /dev/nvme0n1p2此处nvme0n1p2为主分区 mkswap /dev/nvme0n1p3此处nvme0n1p3为交换分区 swapon /dev/nvme0n1p3打开交换分区 2.5挂载分区 mount /dev/nvme0n1p2 /mnt mkdir /mnt/boot mount /dev/nvme0n1p1 /mnt/boot 2.
